 파일 암호화 시스템

## Features

- 직관적인 드래그 앤 드롭 인터페이스
- AES-256 암호화 알고리즘 사용
- 유연한 청크 크기 설정 (1MB ~ 1GB)
- 실시간 진행 상황 모니터링
- 비밀번호 보호 키 파일 시스템
- 모던한 디자인의 UI/UX

## Requirements

| Component | Specification |
|-----------|--------------|
| Java      | 11 or higher |
| Memory    | 4GB RAM      |
| OS        | Windows 10/11, macOS 10.14+, Linux |
| Storage   | 100MB        |
